# MNIST Digit Classifier

## Project Overview
This project is dedicated to building and optimizing a classifier for the MNIST dataset, which includes a large collection of handwritten digits. The aim is to accurately recognize and classify handwritten digits using machine learning techniques.

## Features
- **Dataset Loading**: Load and preprocess the MNIST dataset for machine learning models.
- **Statistical Analysis**: Perform detailed statistical analysis of the dataset, including pixel distribution and class variance.
- **Classification**: Develop a classification model using TensorFlow, complete with detailed metrics and performance optimization.

## Technologies Used
- Python
- TensorFlow
- Jupyter Notebook
